    <h3>Quick Start</h3>
    <ul>
        <li>Clone the repo: <code>git clone https://github.com/pixelspring/springkit.git</code></li>
        <li>Install dependencies with: <code>npm install</code></li>
        <li>Build with the Gulpfile: <code>gulp</code></li>
    </ul>
